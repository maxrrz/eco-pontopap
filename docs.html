Introdução

O EcoPonto é um sistema IoT que utiliza sensores ultrassônicos para medir o nível de enchimento dos contentores. Os dados são enviados para um servidor em tempo real e exibidos em uma interface web intuitiva.

Hardware

Componentes

  • ESP32 (Microcontrolador)
  • Sensor Ultrassônico HC-SR04
  • Display OLED 128x64
  • Bateria de Lítio 3.7V
  • Painel Solar 5V

Esquema de Ligações

ESP32 -> HC-SR04
- GPIO 5 -> TRIG
- GPIO 18 -> ECHO
- 3.3V -> VCC
- GND -> GND

ESP32 -> OLED
- GPIO 21 -> SDA
- GPIO 22 -> SCL
- 3.3V -> VCC
- GND -> GND

Software

Tecnologias Utilizadas

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js, Express
  • Banco de Dados: MongoDB
  • Firmware: Arduino IDE (C++)

API

Endpoints

GET /api/data

Retorna os dados atuais dos sensores

{
    "yellow": 75,
    "green": 45,
    "blue": 30,
    "timestamp": "2024-02-20T15:30:00Z"
}

GET /api/history

Retorna o histórico de leituras

{
    "data": [
        {
            "yellow": 70,
            "green": 40,
            "blue": 25,
            "timestamp": "2024-02-20T15:00:00Z"
        },
        // ...
    ]
}

Instalação

Requisitos

  • Node.js 14+
  • MongoDB 4.4+
  • Arduino IDE 2.0+

Configuração

# Clonar o repositório
git clone https://github.com/seu-usuario/ecoponto.git

# Instalar dependências
cd ecoponto
npm install

# Configurar variáveis de ambiente
cp .env.example .env
nano .env

# Iniciar o servidor
npm start
