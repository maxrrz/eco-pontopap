# Imagens de Instrução para Instalação do PWA

Este diretório deve conter as imagens de instrução para a instalação do PWA em diferentes plataformas.

## Imagens Necessárias

### Para iOS (Safari)
- `ios-share.png` - Captura de tela mostrando o botão de compartilhamento no Safari
- `ios-add.png` - Captura de tela mostrando a opção "Adicionar à Tela de Início"
- `ios-confirm.png` - Captura de tela mostrando o botão de confirmação

### Para Android (Chrome)
- `android-menu.png` - Captura de tela mostrando o menu do Chrome
- `android-install.png` - Captura de tela mostrando a opção "Instalar aplicativo"
- `android-confirm.png` - Captura de tela mostrando o botão de confirmação

### Para Desktop (Chrome/Edge)
- `desktop-icon.png` - Captura de tela mostrando o ícone de instalação na barra de endereço
- `desktop-prompt.png` - Captura de tela mostrando o prompt de instalação

## Recomendações

- As imagens devem ter uma largura entre 300px e 600px para melhor visualização
- Use capturas de tela reais para maior clareza
- Adicione setas ou círculos para destacar os botões importantes
- Salve as imagens em formato PNG para melhor qualidade
